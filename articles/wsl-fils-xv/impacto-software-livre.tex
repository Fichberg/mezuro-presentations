\begin{abstract}
  This resumed article approaches one of the most important decisions when starting a new free software project: to adopt and base the new project in a old and madure one; or to create a whole new and independent one. Will be discussed the vantages and disavantages for each option. For, finally, present a real experience from a project that started by choosing the second option, during it's early development choose to adopt the first one, to  recently again choose the second one.
\end{abstract}
     
\begin{resumo} 
  Este artigo resumido trata sobre uma das decisões mais importantes no momento de se iniciar um novo projeto de software livre: adotar como base outro projeto já maduro; ou escrever um totalmente novo e independente. Serão discutidas brevemente as vantagens e desvantagens de cada opção. Para, por fim, apresentar uma experiência real de um projeto que teve início decidindo pela segunda opção, durante o início de seu desenvolvimento resolveu adotar a segunda opção, para recentemente novamente adotar a primeira opção.
\end{resumo}
